Practizer

English / Deutsch

Practizer is a desktop app to practice with audio recordings. Load a file, change tempo and pitch independently, set markers and A/B loops, use a metronome and a beat grid, and save projects.

Features

  • Independent tempo & pitch via SoundTouch (JNI, LGPL)
  • A/B looping, markers, zoomable waveform
  • Metronome, tap-tempo, beat detection, grid/snap
  • Project files (*.practice.json) with light autosave
  • Optional MP3/FLAC/Vorbis via JavaSound SPI
