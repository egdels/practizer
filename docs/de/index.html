Practizer

English / Deutsch

Practizer ist eine Desktop‑App zum Üben mit Audioaufnahmen. Lade eine Audiodatei, ändere Tempo (Geschwindigkeit) und Tonhöhe (Key) unabhängig voneinander, setze Marker und A/B‑Loops, nutze ein Metronom und ein Beat‑Raster.

Funktionen

  • Tempo- und Tonhöhenänderung (SoundTouch, JNI, LGPL)
  • A/B‑Looping, Marker, Zoom‑bare Wellenform
  • Metronom, Tap‑Tempo, Beat‑Erkennung, Grid/Snap
  • Projektdateien (*.practice.json) mit Autosave
  • Optional: MP3/FLAC/Vorbis via JavaSound SPI
