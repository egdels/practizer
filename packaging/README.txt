Practizer — Desktop Icon Pack
=============================
Contents:
  • Practizer-1024.png — master icon (1024×1024)
  • linux-png/ — PNG icons for Linux (16..512 px)
  • Practizer.ico — multi-size Windows icon (16..256 px)
  • Practizer.iconset/ — macOS iconset folder
  • build-icns.sh — run on macOS to create Practizer.icns

macOS:
  1) On macOS, open a terminal and run:
     ./build-icns.sh
  2) Use Practizer.icns in your .app bundle or installer.

Windows:
  • Use Practizer.ico for your EXE or MSI installer.

Linux:
  • Use PNGs from linux-png/ for your .desktop launcher and themes.
